The following Parser related source files will be generated by Maven (and thus not stored in version control):
Parser.java
ParserConstants.java
ParserTokenManager.java
ParserTreeConstants.java
Token.java
TokenMgrError.java
JJTParserState.java

See pom.xml how this magic works.



To recreate Parser (using src/parser/Parser.jjt), use a command line switch:
mvn -Dparser.create=true package



If there are new structures or nodetypes in the Parser.jjt, generate the new AST nodefiles by using a command line switch:
mvn -Dparser.nodefiles=true package

Maven build will display the instructions.
